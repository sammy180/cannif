# Images Directory

This directory contains all images used in the OpenCAN-ESP32 launch website.

## Directory Structure

- `hero/` - Hero section images and main product visuals
- `features/` - Feature demonstration images and diagrams
- `products/` - Product photos, variants, and technical images
- `logos/` - Company logos, partner logos, and branding assets

## Image Guidelines

- Use high-quality images (minimum 1920px width for hero images)
- Optimize images for web (WebP format preferred)
- Include alt text for accessibility
- Maintain consistent aspect ratios within each category

## Recommended Images

### Hero Section
- Main product photo (3D render or prototype)
- Product in automotive environment
- Close-up of ESP32 board
- CAN bus connection diagram

### Features
- IP68 enclosure demonstration
- CAN FD vs CAN comparison diagram
- Automotive grade certification badges
- ESP32 connectivity features

### Products
- All three variants (CAN, CAN FD, LIN)
- Technical specifications diagram
- Size comparison with other devices
- Installation examples

### Logos
- OpenCAN-ESP32 logo
- ESP32 logo
- Partner/community logos
- Certification badges
